# Sportgaststätte ASV Fürth - Website

Dies ist die offizielle Website für die Sportgaststätte ASV Fürth, entwickelt mit Next.js und modernem CSS.

## Features

- **Startseite**: Willkommensbereich mit Hero-Image.
- **Speisekarte**: Vorschau auf die Mischung aus deutscher und armenischer Küche.
- **Anfahrt**: Integrierte Google Maps Karte und Kontaktinformationen.
- **Responsive Design**: Optimiert für Desktop, Tablet und Mobile.
- **SEO**: Optimierte Metadaten für Suchmaschinen.

## Technologien

- [Next.js](https://nextjs.org/) - React Framework
- [CSS Modules](https://github.com/css-modules/css-modules) - Modulares Styling
- [TypeScript](https://www.typescriptlang.org/) - Typensicherheit

## Starten des Projekts

Um den Entwicklungsserver zu starten:

```bash
npm run dev
```

Öffnen Sie [http://localhost:3000](http://localhost:3000) in Ihrem Browser.

## Anpassung

- **Texte**: Die Inhalte können direkt in den `src/components/` Dateien geändert werden (z.B. `Hero.tsx`, `Map.tsx`).
- **Styles**: Globale Farben und Stile finden sich in `src/app/globals.css`.
- **Bilder**: Aktuell werden Platzhalter von Unsplash verwendet. Diese können durch lokale Bilder im `public/` Ordner ersetzt werden.
